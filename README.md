# Personal Website

My [personal website](https://zhangkai.io), built with Next.js and hosted on GitHub Pages. Forked from [Michael D'Angelo's personal-site](https://github.com/mldangelo/personal-site).

## Local Development

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

Preview the production build locally:

```bash
npm run build
npx serve@latest out
```
